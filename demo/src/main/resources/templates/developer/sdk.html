快速上手指南

三步快速集成简易云SDK

1

安装SDK

使用包管理工具安装SDK，以Node.js为例：

npm install @simple-cloud/sdk
2

初始化客户端

创建SDK客户端实例并配置认证信息：

const { Client } = require('@simple-cloud/sdk');
const client = new Client({ accessKeyId: 'YOUR_ACCESS_KEY_ID', secretAccessKey: 'YOUR_SECRET_ACCESS_KEY', region: 'ap-east-1' });
3

调用云服务API

使用客户端实例调用云服务API：

async function createServer() { try { const result = await client.cvm.createInstance({ instanceName: 'my-server', instanceType: 'n1.standard.1', imageId: 'img-12345678' }); console.log('服务器创建成功:', result); } catch (error) { console.error('创建失败:', error); } }
createServer();
